    （二）Docker-帮助启动类命令

    mahy 2023/6/21 docker

    帮助启动类命令，关于docker的启动、停止、状态和帮助

    # 1. 启动docker

    systemctl start docker
    1

    # 2. 停止docker

    systemctl stop docker
    1

    # 3. 重启docker

    systemctl restart docker
    1

    # 4. 查看docker状态

    systemctl status docker
    1

    # 5. 开机启动

    systemctl enable docker
    1

    # 6. 查看docker概要信息

    docker info
    1

    # 7. 查看帮助文档

    docker --help
    1

    # 8. 查看命令帮助文档

    docker [具体命令] --help
